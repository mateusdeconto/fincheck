# FinCheck — Roadmap

## Phase 1 — MVP Funcional ✅ (atual)

**Goal:** Usuário consegue completar o fluxo do início ao fim e receber diagnóstico via IA.

### Plans
1. **Backend API** — Express + rotas de diagnóstico (streaming) e chat
2. **Frontend Base** — Vite + React + Tailwind, roteamento por estado
3. **Componentes do Fluxo** — Onboarding, Questionnaire, Loading, Diagnosis, Chat

### Success Criteria
- [ ] Fluxo completo funciona end-to-end
- [ ] Diagnóstico aparece progressivamente (streaming)
- [ ] Chat responde com contexto dos dados
- [ ] Funciona no mobile

## Phase 2 — Polimento (futura)

- Animações e microinterações
- Exportar diagnóstico como PDF
- Compartilhar diagnóstico via link

## Phase 3 — Produto (futura)

- Autenticação e histórico de diagnósticos
- Comparativo mês a mês
- Integração com planilhas/contabilidade
